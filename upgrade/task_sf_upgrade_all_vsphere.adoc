= Mettez à niveau vos composants vSphere pour un système de stockage NetApp SolidFire grâce au plug-in Element pour vCenter Server.
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
Vous devez effectuer des étapes supplémentaires lors de la mise à niveau des composants VMware vSphere. Ces étapes sont nécessaires pour les systèmes dotés du plug-in Element pour vCenter Server.

.Étapes
. Pour les mises à niveau vCSA, https://docs.netapp.com/us-en/vcp/vcp_task_qossioc.html#clear-qossioc-settings["clair"^] Paramètres QoSSIOC dans le plug-in (* Configuration de NetApp Element > Paramètres QoSSIOC *).  Le champ *État QoSSIOC* s'affiche `Not Configured` une fois le processus terminé.
. Pour les mises à niveau vCSA et Windows, https://docs.netapp.com/us-en/vcp/task_vcp_unregister.html["désinscription"^] le plug-in du serveur vCenter auquel il est associé à l'aide de l'utilitaire d'enregistrement.
. https://docs.vmware.com/en/VMware-vSphere/6.7/com.vmware.vcenter.upgrade.doc/GUID-7AFB6672-0B0B-4902-B254-EE6AE81993B2.html["Mettez à niveau vSphere, y compris vCenter Server, ESXi, les machines virtuelles et les autres composants VMware."^].
+
[IMPORTANT]
====
Vous devriez mettre à niveau le plug-in NetApp Element pour vCenter Server vers la version 5.0 ou ultérieure afin de pouvoir déployer le plug-in avec VMware vCenter 7.0 Update 3 sans avoir à appliquer de solution de contournement.

Lors de la mise à niveau vers VMware vCenter Server 7.0 Update 3, le déploiement du plug-in Element pour vCenter Server 4.x échoue. Pour résoudre ce problème avec Spring Framework 4, consultez https://kb.netapp.com/Advice_and_Troubleshooting/Hybrid_Cloud_Infrastructure/NetApp_HCI/vCenter_plug-in_deployment_fails_after_upgrading_vCenter_to_version_7.0_U3["cet article de la base de connaissances"^].

====
. https://docs.netapp.com/us-en/vcp/vcp_task_getstarted.html#register-the-plug-in-with-vcenter["Registre"^]Le plug-in Element pour vCenter Server est de nouveau disponible avec vCenter.
. https://docs.netapp.com/us-en/vcp/vcp_task_getstarted.html#add-storage-clusters-for-use-with-the-plug-in["Ajouter des clusters"^]en utilisant le plug-in.
. https://docs.netapp.com/us-en/vcp/vcp_task_getstarted.html#configure-qossioc-settings-using-the-plug-in["Configurer les paramètres QoSSIOC"^]en utilisant le plug-in.
. https://docs.netapp.com/us-en/vcp/vcp_task_qossioc.html#enabling-qossioc-automation-on-datastores["Activer QoSSIOC"^]pour tous les magasins de données contrôlés par le plug-in.
