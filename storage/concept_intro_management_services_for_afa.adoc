= Services de gestion pour le stockage 100% flash SolidFire
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Les services de gestion offrent des fonctionnalités de gestion centralisées et étendues pour le stockage 100 % flash SolidFire .  Ces services comprennent la télémétrie du système Active IQ , la journalisation et les mises à jour de service, ainsi que le service QoSSIOC pour le plug-in Element pour vCenter.
