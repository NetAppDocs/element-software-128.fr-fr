= Configurer les paramètres du cluster
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Vous pouvez consulter et modifier les paramètres globaux du cluster et effectuer des tâches spécifiques au cluster depuis l'onglet Cluster de l'interface utilisateur Element.

Vous pouvez configurer des paramètres tels que le seuil de remplissage du cluster, l'accès au support, le chiffrement au repos, les volumes virtuels, SnapMirror et le client de diffusion NTP.
