= Configurer un système de stockage Element
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
La mise en place d'un système de stockage logiciel NetApp Element implique la configuration d'un nœud de stockage, la création d'un cluster de stockage et l'ajout de disques au cluster.  Si vous utilisez un réseau Fibre Channel, vous pouvez configurer un nœud Fibre Channel.
