= Authentification API
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Vous pouvez vous authentifier auprès du système lors de l'utilisation de l'API en incluant un en-tête d'authentification HTTP Basic dans toutes les requêtes API.  Si vous omettez les informations d'authentification, le système rejette la requête non authentifiée avec une réponse HTTP 401.  Le système prend en charge l'authentification HTTP Basic via TLS.

Utilisez le compte d'administrateur du cluster pour l'authentification API.
