= GetOrigin
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Vous pouvez utiliser le `GetOrigin` méthode pour obtenir le certificat d'origine du lieu de construction du nœud.



== Paramètres


NOTE: Cette méthode renvoie « null » en l'absence de certification d'origine.

Cette méthode ne requiert aucun paramètre d'entrée.



== valeur de retour

Cette méthode permet de récupérer les informations de certification d'origine du fournisseur.



== Exemple de demande

Les requêtes pour cette méthode sont similaires à l'exemple suivant :

[listing]
----
{
 "method": "GetOrigin",
 "id" : 1
}
----


== Exemple de réponse

Cette méthode renvoie une réponse similaire à l'exemple suivant :

[listing]
----
{
  "integrator": "SolidFire",
  "<signature>": {
    "pubkey": [public key info],
    "version": 1,
    "data": [signature info]
  },
   "contract-id": "none",
   "location": "Boulder, CO",
   "organization": "Engineering",
   "type": "element-x"
  }
 ]
}
----
