= ListQoSPolicies
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Vous pouvez utiliser le `ListQoSPolicies` Méthode permettant de lister les paramètres de toutes les politiques QoS du système.



== Paramètres

Cette méthode ne requiert aucun paramètre d'entrée.



== Valeurs de retour

Cette méthode renvoie les valeurs suivantes :

|===
| Nom | Description | Type 


 a| 
Politiques de QoS
 a| 
Liste détaillée de chaque politique QoS.
 a| 
xref:reference_element_api_qospolicy.adoc[Politique de qualité]tableau

|===


== Exemple de demande

Les requêtes pour cette méthode sont similaires à l'exemple suivant :

[listing]
----
{
  "id": 231,
  "method": "ListQoSPolicies",
  "params": {}
}
----
