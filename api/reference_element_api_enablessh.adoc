= ActiverSsh
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Vous pouvez utiliser le `EnableSsh` méthode pour activer le service Secure Shell (SSH) pour un seul nœud.  Cette méthode n'affecte pas la durée du délai d'expiration SSH à l'échelle du cluster et n'exempte pas le nœud de la désactivation du SSH par le délai d'expiration SSH global.



== Paramètre

Cette méthode ne comporte aucun paramètre d'entrée.



== valeur de retour

Cette méthode a la valeur de retour suivante :

|===
| Nom | Description | Type 


 a| 
activé
 a| 
État du service SSH pour ce nœud.
 a| 
booléen

|===


== Exemple de demande

Les requêtes pour cette méthode sont similaires à l'exemple suivant :

[listing]
----
{
  "method": "EnableSsh",
  "params": {
     },
  "id" : 1
}
----


== Exemple de réponse

Cette méthode renvoie une réponse similaire à l'exemple suivant :

[listing]
----
{
  "id" : 1,
  "result" : {"enabled": true}
}
----
