= serveur de journalisation
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
L'objet loggingServer contient des informations sur tous les hôtes de journalisation configurés pour le cluster de stockage.  Vous pouvez utiliser `GetRemoteLoggingHosts` pour déterminer quels sont les hôtes de journalisation actuels, puis utiliser `SetRemoteLoggingHosts` pour définir la liste souhaitée des hôtes de journalisation actuels et nouveaux.



== membres de l'objet

Cet objet contient les membres suivants :

|===
| Nom | Description | Type 


 a| 
hôte
 a| 
Adresse IP du serveur de journalisation.
 a| 
chaîne



 a| 
port
 a| 
Numéro de port utilisé pour communiquer avec le serveur de journalisation.
 a| 
entier

|===
