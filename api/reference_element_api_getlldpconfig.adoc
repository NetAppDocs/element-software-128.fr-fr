= ObtenirLldpConfig
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Vous pouvez utiliser le `GetLldpConfig` méthode pour obtenir la configuration du protocole LLDP (Link Layer Discovery Protocol) pour chaque nœud d'un cluster de stockage.



== Paramètres

Cette méthode ne requiert aucun paramètre d'entrée.



== Valeurs de retour

Cette méthode renvoie les valeurs suivantes :

|===
| Nom | Description | Type 


 a| 
lldpConfig
 a| 
Informations concernant la configuration LLDP du cluster de stockage.
 a| 
objet JSON

|===


== Exemple de demande

Les requêtes pour cette méthode sont similaires à l'exemple suivant :

[listing]
----
{
   "method": "GetLldpConfig",
   "id" : 1
}
----


== Exemple de réponse

Cette méthode renvoie une réponse similaire à l'exemple suivant :

[listing]
----
{
	"id": null,
		"result": {
			"lldpConfig": {
				"enableLldp": true,
				"enableMed": false,
				"enableOtherProtocols": true
		}
	}
}
----
