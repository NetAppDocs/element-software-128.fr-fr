= règle de politique snapMirror
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
L'objet snapMirrorPolicyRule contient des informations sur les règles d'une politique SnapMirror .



== membres de l'objet

Cet objet contient les membres suivants :

|===
| Nom | Description | Type 


 a| 
snapMirrorLabel
 a| 
Étiquette de copie instantanée, utilisée pour la sélection de la copie instantanée dans les relations de protection des données étendues.
 a| 
chaîne



 a| 
garder le compte
 a| 
Spécifie le nombre maximal de copies d'instantané conservées sur le volume de destination SnapMirror pour une règle.
 a| 
entier

|===
