= Méthodes d'API des groupes d'accès au volume
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Les méthodes de groupe d'accès aux volumes vous permettent d'ajouter, de supprimer, d'afficher et de modifier des groupes d'accès aux volumes, qui sont des ensembles de volumes auxquels les utilisateurs peuvent accéder à l'aide d'initiateurs iSCSI ou Fibre Channel.

* xref:reference_element_api_deletevolumeaccessgroup.adoc[Supprimer le groupe d'accès au volume]
* xref:reference_element_api_listvolumeaccessgroups.adoc[Lister les groupes d'accès aux volumes]
* xref:reference_element_api_removevolumesfromvolumeaccessgroup.adoc[Supprimer les volumes du groupe d'accès aux volumes]
* xref:reference_element_api_removeinitiatorsfromvolumeaccessgroup.adoc[Supprimer les initiateurs du groupe d'accès aux volumes]
* xref:reference_element_api_modifyvolumeaccessgroup.adoc[Modifier le groupe d'accès au volume]
