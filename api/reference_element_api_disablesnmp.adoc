= Désactiver SNMP
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Vous pouvez utiliser le `DisableSnmp` Méthode pour désactiver SNMP sur les nœuds du cluster.



== Paramètre

Cette méthode ne comporte aucun paramètre d'entrée.



== valeur de retour

Cette méthode ne renvoie aucune valeur.



== Exemple de demande

Les requêtes pour cette méthode sont similaires à l'exemple suivant :

[listing]
----
{
  "method": "DisableSnmp",
  "params": {},
  "id" : 1
}
----


== Exemple de réponse

Cette méthode renvoie une réponse similaire à l'exemple suivant :

[listing]
----
{
  "result" : {},
  "id" : 1
}
----
