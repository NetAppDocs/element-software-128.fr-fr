= Initiateurs
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Les initiateurs permettent aux clients externes d'accéder aux volumes d'un cluster, servant de point d'entrée pour la communication entre les clients et les volumes.  Vous pouvez utiliser des initiateurs pour un accès aux volumes de stockage basé sur CHAP plutôt que sur un compte.  Un seul initiateur, lorsqu'il est ajouté à un groupe d'accès aux volumes, permet aux membres du groupe d'accéder à tous les volumes de stockage ajoutés au groupe sans nécessiter d'authentification.  Un initiateur ne peut appartenir qu'à un seul groupe d'accès.
